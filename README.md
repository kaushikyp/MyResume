# Phani Kaushik EA-style resume website

GitHub-ready and Vercel-ready React project.

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Deploy to Vercel
1. Upload this folder to a GitHub repo.
2. Import that repo in Vercel.
3. Vercel should auto-detect it as a Vite project.
4. Click Deploy.

## Main files
- src/App.jsx
- src/main.jsx
- src/index.css
- vite.config.js
- package.json
